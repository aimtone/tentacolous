Tentacolous

Run Java methods when your database changes, even when the change did not come from your application.

Tentacolous gives Spring Boot projects a simple way to react to real PostgreSQL INSERT, UPDATE and DELETE operations using triggers, an event table and annotated listeners. The latest stable release is io.github.aimtone:tentacolous:0.1.6 from Maven Central.
